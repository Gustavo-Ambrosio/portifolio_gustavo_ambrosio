# 🚀 Gustavo Ambrosio – Portfolio

Professional portfolio website built for GitHub Pages.

**Live at:** [https://gustavo-ambrosio.github.io/Portif%C3%B3lio/](https://gustavo-ambrosio.github.io/Portif%C3%B3lio/)

## Stack
- Pure HTML5, CSS3, Vanilla JavaScript
- No frameworks or dependencies (except Google Fonts)
- Fully responsive and mobile-friendly

## Sections
- **Hero** – Animated introduction with typing effect and floating particles
- **About** – Professional summary and contact links
- **Skills** – Technical stack organized by category
- **Experience** – Animated timeline of professional history
- **Projects** – GitHub repositories showcase
- **Education & Certifications** – Academic background and language proficiency
- **Contact** – All contact channels

## Deploy to GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your site will be live at `https://gustavo-ambrosio.github.io/<repo-name>/`
